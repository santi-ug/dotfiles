# Hyper Key Bindings

> **Hyper** = `CapsLock` — tap alone sends `Escape`

---

## Space

| Keys | Action |
|------|--------|
| `Hyper + Space` | Quick Add Reminder |

## B — Browse

| Keys | Action |
|------|--------|
| `Hyper + B + T` | Twitter |
| `Hyper + B + P` | Calendar (mxstbr.com/cal) |
| `Hyper + B + Y` | Hacker News |

| Keys | Action |
|------|--------|
| `Hyper + O + B` | Orion |
| `Hyper + O + C` | Calendar |
| `Hyper + O + V` | Zed |
| `Hyper + O + K` | Slack |
| `Hyper + O + N` | Notion |
| `Hyper + O + T` | WezTerm |
| `Hyper + O + F` | Finder |
| `Hyper + O + S` | Spotify |
| `Hyper + O + W` | WhatsApp |
| `Hyper + O + R` | Reminders |
